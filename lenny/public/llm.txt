# Lenny for Libraries

> Lenny is a plug-and-play, open-source, Library-in-a-Box that empowers libraries to preserve, own, and lend digital books on their own terms.

## About

Lenny is an open-source digital library server that enables libraries to:
- Host and preserve digital book collections
- Lend ebooks using OPDS (Open Publication Distribution System)
- Sync metadata from OpenLibrary.org automatically
- Support LCP (Library of Patron Copies) DRM
- Integrate with mobile reading apps like Thorium Reader

## Quick Installation

```bash
curl -fsSL https://raw.githubusercontent.com/ArchiveLabs/lenny/refs/heads/main/install.sh | sudo sh
```

## Key Features

- **Preloaded Books**: Thousands of open access titles included out of the box
- **Bookshelf**: Securely store and host millions of digital books
- **Card Catalog**: Auto-sync metadata from OpenLibrary
- **Lending System**: Configurable lending rules with LCP support
- **Mobile Apps**: Load your Lenny feed on any OPDS-compatible reader
- **Marketplace**: Expand your collection with ease

## Deployment Options

1. **Self-Hosted**: Run Lenny as a Docker container on your own server
2. **Plug & Play**: Pre-installed on Raspberry Pi (coming soon)
3. **Hosted**: Managed hosting with web dashboard (coming soon)
